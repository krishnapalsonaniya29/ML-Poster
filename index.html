Multi-Modal Machine Learning Framework for Intelligent Data Analysis
Team Members: Krishnapal Sonaniya (0801IT231069 / krishnapalsonaniyasgsits@gmail.com) | Piyush Soni (0801IT231092 / piyushsoni8839@gmail.com) | Mohit Yadav (0801IT231079 / yadavmohit231079@gmail.com)
Mentor: Dr. K.K. Sharma | Dr. Lalit Purohit | Dr. Upendra Singh | Mr. Akshay Gupta
Objective
Design and implement a unified multi-modal machine learning framework capable of handling Image, Text, Numeric, Audio, and Video data. The system aims to apply modality-specific preprocessing and model selection to achieve high accuracy while maintaining scalability and real-world usability.
Methodology
Each modality follows a dedicated pipeline: data ingestion → preprocessing (TF-IDF, MFCC, frame extraction, scaling) → feature transformation → model training → evaluation → best model selection. The final system integrates all pipelines into a single interface for consistent and efficient inference.
Models Used
A variety of models were selected based on data characteristics, including CNN and MobileNetV2 for images and video, SVM and Logistic Regression for text and numeric data, and LSTM-based architectures for sequential learning. Each model was optimized using appropriate preprocessing techniques.
Key Results
The framework achieved strong performance across all modalities, with highest accuracy in numeric classification (97.5%) and consistently high results in text, audio, and video tasks. The study demonstrates that modality-specific model selection significantly improves performance over a generalized approach.

RAVDESS Emotional Speech Dataset
Type Audio Data (Multi-class Classification) Classes Neutral, Calm, Happy, Sad, Angry, Fearful, Disgust, Surprised Input MFCC (40 × 128), ~3 sec audio
Preprocessing MFCC Extraction (Librosa), Padding/Truncation (128 steps)

Feature Representation MFCC features capture the spectral characteristics of audio signals, converting sound into a visual-like representation. This enables CNN models to effectively learn spatial patterns similar to image-based inputs.
